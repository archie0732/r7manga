## 功能說明

R7manga 包含 多個網站的漫畫與小說，全站內容皆為免費


1. 支援多種網站，無需來回切換網站來觀看各種書籍
2. 限制集內容擁有保護開關，避免在公共場合誤觸設社死
3. 關聯網站搜索，支援不同網站同關鍵詞搜索，讓尋找作品無負擔
4. 跨網站查詢相關漫畫

- 網站

|支援網站|支援功能|備註|
|---|---|-|
|看漫畫 / 漫畫櫃 (manhuagui)|查詢、觀賞、收藏、追蹤作者||
|嗶哩嗶哩輕小說 (linovelib)|查詢、觀賞、收藏、追蹤作者|開發中|
|nhentai|查詢、觀賞、收藏、追蹤作者||
|紳士漫畫 wnacg|查詢、觀賞、收藏、追蹤作者|可以使用，部分功能還在熬夜製作|
|hentaipaw|查詢、觀賞、收藏、追蹤作者|開發中|

## 如何在本地運行

1. 首先，請確保你的電腦上已安裝 `bun` 或者 `npm` 

>下面將使用 bun 來做演示

[bun 官網](https://bun.sh/)

2. 安裝依賴

```bash
bun install
```

3. 設定環境變數，請見 環境變數 章節

4. 啟動

```
bun run dev
```

### 環境變數

請在根目錄建立 `.env` 檔案，並參考 `.env.example` 設定：
| 變數名稱 | 說明 | 範例值 |
|---|---|---|
| NEXT_PUBLIC_API_URL | 後端 API 地址 | http://localhost:3000 |
| MONGODB_URI | (若有資料庫) | mongodb://... |
